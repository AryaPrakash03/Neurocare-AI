NeuroCare AI

Brain Tumor Detection and Classification using Deep Learning

Overview

NeuroCare AI is a deep learning-based system for detecting and classifying brain tumors from MRI images. Using VGG16 transfer learning, the system achieves over 95% accuracy in classifying tumor vs non-tumor scans.

Objectives

  • Automate tumor detection from MRI images
  • Classify scans into tumor / non-tumor categories
  • Provide an assistive tool for early diagnosis
